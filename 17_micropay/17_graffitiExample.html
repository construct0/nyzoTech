Graffiti example

This page currently allows simple drawing on the wall, but nothing is sent to the blockchain.

This page will have a virtual graffiti wall on which users can draw using automatic transactions. The wall will have a resolution that is dynamically set by the image data provided from https://relay0.nyzo.co/graffiti.json.

This page is in active development to demonstrate a realtime multiuser application on the Nyzo blockchain.
