System design

Micropay currently supports two use cases: sending tips and paying for content. A third use case, playing games, is currently in development.

Playing games

Extended Micropay functionality for playing games is currently in development. For games, users will pre-approve a maximum amount to be used for the transactions that will record game states. The web page will then generate the necessary transaction information in the course of game play, and it will interact with the Chrome extension or the mobile application to produce and sign the necessary transactions.
