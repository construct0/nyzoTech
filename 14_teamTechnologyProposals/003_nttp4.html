NTTP-4: WordPress plugin

The Nyzo Chrome extension allows tips to be sent and digital content to be purchased.

NTTP-4: we propose developing a WordPress plugin to complement the Chrome extension. This plugin would allow users of WordPress to easily include Nyzo tip buttons and Micropay elements on their sites.

The plugin would be released into the public domain with the Unlicense, and it would be made available on GitHub.
